L'internationalisation I18N

Hello Angular

Angular is the Best Framework in the World !

0- ng add @angular/localize
Cette dépendance ajoutel support de l'internationalisation I18N à notre projet (NG>=9)

1- Marquer les zones (éléments) à traduire avec i18n

2- Editer le fichier angular.json ===> 3 zones Projets (messages.xlf) - build - serve

4- Générer le fichier xlf (format json)
ng extract-xi18n --output-path src/locale

5- Tester la traduction
ng serve --configuration=fr --open
ng serve --configuration=es --open

6- Build final
ng build --localize
